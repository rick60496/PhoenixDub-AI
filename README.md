# PhoenixDub-AI
PhoenixDub AI é um estúdio de dublagem e edição inteligente para PCs modestos. Usa IAs leves (Chatterbox/Gema) para dublar jogos e vídeos localmente. Por ser Beta e focado em PCs fracos, a tradução tem limites e erros podem ocorrer. Futuramente, evoluirá para um ecossistema completo de ferramentas de edição guiadas por IA.
